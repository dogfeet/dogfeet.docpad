Google Web Font Api에 보면 여러가지 폰트들이 있지만 적당한 코딩용 한글 폰트가 정말 아쉽습니다.

개인적으로 네이버 코딩 폰트를 즐겨 사용하고 있는데 그걸 웹에서 사용하려니까 네이버가 아직 웹에 올려서 공개하지 않았습니다.

음 이 폰트라도 네이버가 만든 것을 사용하고 있으니까 저도 네이버 유저입니다.

이 블로그는 코딩 블로그니까 폰트도 코딩폰트를 사용하는 것이 적당해 보여서 폰트를 네이버코딩폰트를 적용했습니다.

방법은 네이버코딩폰트를 다운받아서, ttf를 웹폰트 포멧인 woff로 변환합니다. woff 변환도구인 sfnt2woff를 사용할 수 있습니다. woff파일을 만들었으면 어딘가 올려두고 서비스해야 합니다.

이 블로그를 위해 이파일들을 티스토리에 올렸습니다. 네이버가 만든 폰트를 다음의 서버에 올려두고 텀블러에서 사용한다는 전략입니다. 이 모든게 폰트만 너무 잘 만든 멍충이 탓입니다.

티스토리 블로그의 skin올리는 곳에 파일을 .png로 확장자를 바꿔서 올렸습니다.ㅋㅋㅋ woff 확장자로 파일에 접근할 수 없었습니다.

Google Web Font Api에 보면 잘 설명되 있지만 font-face를 만들어 줍니다.

#!css
@font-face {
  font-family: 'NaverCoding';
  font-style: normal; 
  font-weight: normal;
  src: local('NaverCoding'), url('http://ts.daumcdn.net/{중간생략}/coding.png') format('woff');
}

물론 이 파일도 티스토리에 올렸습니다.—;;; 이 스타일을 넣어주면 NaverCoding이라는 폰트를 사용할 수 있습니다…

추가

이 블로그에서는 네이버 코딩 폰트를 제거 했습니다. 폰트파일이 1MB정도 되서 느리다는 의견이 있습니다. 그래서 Google Web Fonts의 Droid Sans Mono를 code 블럭에만 사용하기로 했습니다.

모빌리스에서 한국어 웹 폰트 api를 운영중입니다. 네이버의 웹폰트는 아닌 것 같은데 어쨓든 나눔글꼴을 사용할 수 있습니다.

http://api.mobilis.co.kr/webfonts/
