기계의 힘을 빌어서라도 순간에 집중하고 싶어요.

네이버, 다음, 네이트등의 포털은 원래 사용하지 않습니다만, 그래도 자꾸 인터넷만 들어가면 딴짓을 하게 됩니다.

의지는 좀 박약하니까—; 기계의 힘을 빌어야죠.

막는 방법

Chrome

주로 크롬을 사용하니까. 일단 크롬만 정리합니다. 굳이 확인해봐야 하는 경우는 다른 브라우저를 사용하면 되니까 문제 없습니다. 그리고 구글이 Custom css & javascript에 대한 Cloud도 제공해주면 좋을 텐데…javascript는 아직 content script라는 것 밖에 못찾았습니다. 그리고 사용법도 좀 들여다 봐야 하는 수준이라…

  • css::window - ${HOME}\AppData\Local\Google\Chrome\User Data\Default\User StyleSheets\Custom.css를 편집합니다.
  • css::mac - ${HOME}/Library/Application Support/Google/Chrome/Default/User StyleSheets/Custom.css를 편집합니다.
  • extension::Temporary Site Blocker - 무의식적으로 입력하는 사이트를 아예 막아버립니다. 쉽게 onoff할 수 있어 편리합니다.
  • extension::FlashBlock - 플래시 막기
  • extionsion::adsweep - 악질 광고는 막아주는 것 같습니다. 그러나 한국 사이트들은 예외인 것 같아요.

막을 것

www.hani.co.kr

무언가 하려고 브라우저를 띄우면 hani를 하고 있는 나를 발견하곤 합니다. Temporary Site Blocker로 차단했습니다.

Google

관련검색은 씨불, 관련떡밥인 듯합니다. 구굴이 관련검색과 HOT 토픽을 추가하는 날 울고 싶었습니다. 손으로 막아야 한다는 귀찬음에 울고 싶었어요.

구글 검색시 나오는 관련검색, HOT 토픽을 제거합니다:

#!css
/* google search */
#rcnt > #rhscol > #rhs { 
    display: none !important;
}

#rcnt > #rhscol > #rhs는 어디 까지나 다른 사이트에서는 이런 dom구조를 갖지 않을 거라고 가정하는 것입니다. 설마 같은 이름을 쓰진 않길.—;;

Daum

다음 Mail은 일관련으로 gmail이 안되도록 blacklist에 추가되는 업체들 때문에 사용합니다. 로그인과 관련된 화면에서 많은 정보를 보여줘서 제가 획가닥 가버립니다. 사실 잘 안써요—;;

/* daum portal */
#wrap > #content > :not(#my), #wrap > #content > #my > :not(#logon) { 
    display: none !important;
}
