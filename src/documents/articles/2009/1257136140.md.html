HBox의 장점은 수평적으로 위치를 자동으로 계산해준다는 점에 있습니다. 그러니 매우 편리한 컨테이너입니다. 하지만 동적으로 계산되는 컴포넌트들을 수정(조정?)하고자 했더니 관련글을 찾을 수가 없었습니다. 인터넷을 뒤지니 찾기가 쉽지 않았습니다. 그래서 방향을 전환하여 소스를 조금 까봤더니 금방 찾을 수 있더군요.

다음과 같은 컴포넌트가 있다고 합시다.

<HBox>
    <A id="a"/>
    <B id="b"/>
    <C id="c"/>
</HBox>

사용자 권한에 따라 HBox에 있는 하위 컴포넌트를 숨기고 싶습니다. 슈도코드는 다음과 같습니다.  

if( 관리자 ) {
    A를 보여준다.
}else{
    A를 숨긴다.
}

 실제 ActionScript코드는 다음과 같습니다.  

if( 관리자 ) {
    a.includeInLayout = true;
    a.visible = true;
}else{
    a.includeInLayout = false;
    a.visible = false;
}

includeInLayout값이 false면 layout시 제외됩니다. 하지만 화면에 보이기 때문에 visible도 같이 false로 해주어야 합니다.
