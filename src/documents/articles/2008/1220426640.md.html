Grails의 config.groovy는 runtime에 evaluate 되는 것이 아니라 ConfigSluper()를 통해 structured data가 된다. ConfigSlurper를 통해 구조화된 데이터를 쉽게 작성할 수 있다.http://groovy.codehaus.org/ConfigSlurper

def config = new ConfigSlurper().parse('''log4j {    appender.stdout = "org.apache.log4j.ConsoleAppender"    appender."stdout.layout"="org.apache.log4j.PatternLayout"    rootLogger="error,stdout"    logger {        org.springframework="info,stdout"    }    additivity {        org.springframework=false    }}''') println configprintln config.log4j.logger

결과

["log4j":["appender":["stdout":"org.apache.log4j.ConsoleAppender", "stdout.layout":"org.apache.log4j.PatternLayout"], "rootLogger":"error,stdout", "logger":["org":["springframework":"info,stdout"]], "additivity":["org":["springframework":false]]]]["org":["springframework":"info,stdout"]]
